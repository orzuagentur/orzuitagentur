# AI Agent Rules

## Project

Luxury futuristic IT agency website for OrzuIT.

The website must feel:

* cinematic
* premium
* futuristic
* technological
* luxury
* powerful

The experience should feel like a next-generation AI company.

---

## Stack

* Next.js
* TypeScript
* Tailwind CSS
* shadcn/ui
* Supabase
* Vercel
* Resend
* Telegram Bot API

---

## Frontend Rules

* use modern premium UI
* use responsive design
* mobile-first
* use smooth animations
* use modular components
* use App Router
* use TypeScript only
* no inline styles
* use reusable sections
* use clean architecture

---

## Design Direction

* dark luxury design
* cinematic gradients
* glassmorphism
* futuristic 3D feel
* smooth scroll
* animated reveals
* premium typography
* strong visual hierarchy
* glowing accents
* modern AI-company style

---

## Animation Rules

* animations must feel premium
* smooth transitions
* subtle hover effects
* section reveal animations
* floating orbs
* gradient motion
* motion-safe animations
* support prefers-reduced-motion

---

## Backend Rules

* use Supabase for database
* use Resend for emails
* use Telegram bot notifications
* use secure server actions
* use environment variables
* separate frontend and backend logic

---

## Admin Panel Rules

Admin dashboard must allow editing:

* hero text
* services
* portfolio
* testimonials
* contacts
* footer
* SEO
* navigation
* analytics
* settings

Admin panel must feel like:

* modern SaaS dashboard
* premium control center
* scalable CMS system

---

## Code Quality

* reusable components only
* clean folder structure
* avoid duplicated code
* scalable architecture
* production-ready code
* accessible UI
* SEO-friendly structure
